# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Overview

This is a **documentation-only repository** — there is no application code, build system, or test suite. It contains the TechAccel Framework (TAF), a structured methodology for technology adoption developed as part of the MIT Technology Leadership Program (TLP) Cohort 2024/2025.

## Structure

- `techaccel_framework.md` — The core framework document defining TAF's four stages: Technology Scouting and Assessment, Feasibility Analysis, Pilot Implementation, and Validation and Scaling.
- `CONTRIBUTING.md` — Contribution guidelines including branching workflow, PR process, and style expectations.
- `images/` — Contributor photos referenced by README.

## Contributing Workflow

1. Fork and clone the repo
2. Create a feature branch (`git checkout -b your-branch-name`)
3. Edit markdown files following the style guide (standard Markdown, clear professional language, PEP8 for any code examples)
4. Open a Pull Request with a detailed description referencing related issues

## Style Guide

- Follow standard Markdown formatting conventions
- Write clear, concise, professional language
- For any code examples, follow common practices for the relevant language (e.g., PEP8 for Python)
